Intro

The movies dataset is available from Kaggle and contains details about 6820 different movies. The following columns are included:

  • budget: the budget of a movie. Some movies dont have this, so it appears as 0
  • company: the production company
  • country: country of origin
  • director: the director
  • genre: main genre of the movie
  • gross: revenue of the movie
  • name: name of the movie
  • rating: rating of the movie (R, PG, etc.)
  • released: release date (YYYY-MM-DD)
  • runtime: duration of the movie
  • score: IMDb user rating
  • star: main actor/actress
  • votes: number of user votes
  • writer: writer of the movie
  • year: year of release

This data has been placed in a Teradata DB for further analysis.

tidypredict

Use the tidypredict package to run predictions against data in the database.

The tidypredict package is used to run predictions inside a database. It does this by taking an R model and translating the object into appropriate SQL commands. The following models are supported:

  • Linear Regression
  • Generalized Linear Model
  • Random Forest
  • MARS models
  • XGBoost models
  • Cubist models
  • Tree models

Build simple linear model to predict movie gross

# Local copy of the data
movies_lcl <- collect(movies_tbl)

# Fit model
model <- linear_reg() %>% 
  set_engine("lm") %>% 
  fit(gross ~ budget + runtime + rating + genre + company, data = movies_lcl)
